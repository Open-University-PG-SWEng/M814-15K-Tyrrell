REQ-ID: NFR-004
Description: The interface shall be available in a number of languages. 
If the language is available, the interface shall use either the language from where the user's IP is situated, or the language the user recorded in his/her preferences, or English (default).
Type: Usability and Humanity, Internationalization (11).
Rationale: The product is intended for an international market. 
Fit Criterion: Given 100 simulated IP adresses, the system shall resolve the IP's geographic location and set the interface to the correct language, or English if no translation is available, unless the language is overriden in the test user's preferences.
Originator: Alvaro Ortiz
Comments: [Alvaro Ortiz] Relates to REQ-07.
